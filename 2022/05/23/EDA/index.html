Exploratory Data Analysis || 探索性数据分析

Created2022-05-23|Updated2022-06-10|Data Analysis
|Word count:429|Reading time:1min|Post View:

Part 1. What is EDA?

  • EDA is an approach for data analysis that employs a variety of techniques to
  1. maximize insight into a data set; uncover underlying structure; detect outliers and anomalies
  2. extract important variables;
  3. test underlying assumptions;
  4. develop parsimonious models;
  5. determine optimal factor settings;

Part 2. The objections of EDA

  • 了解分布： 多数情况下，数据分析的全部工作都是为了量化分布的信息。EDA是第一步，定性的了解分布的信息；建模是第二部，定量的了解分布的信息 + 实现干预。所以从这个角度来看，EDA的工作可以知道我们去建模。

    在EDA中，我们关心的分布常常包括：

    • 单变量整体分布 （通常是histgram
      • 告诉我们单变量分布的信息，对变量有一个初步认识。但是在机器学习中，通常作用不大
    • 多变量|联合分布 （通常是二维， pairplot）：可以直观的看出
      • 通常作用不大
      • pairplot可以提供$P(Y|X)$的信息，但是它其实是一种条件分布。
    • 多变量|条件分布 （通常是二维分布或单变量分布 + 颜色）：
      • 作用更多
    • 时序图
      • 时序和分布有什么区别联系呢？ 一个按时间排序数据，一个按数值排序数据。

Part 2. EDA Assumptions

2.1 Underlying Assumptions

  • Fixed Location: 均值
  • Fixed Variation： 波动
  • Randomness： 时序上独立同分布

if the randomness assumption does not hold, then

  1. all of the usual statistical tests are invalid,
  2. the parameter estimations, calculations, models (all the statistical-related outputs) are meaningless

why not randomness:

  1. autocorrelation (one specific and commom type)
  • Fixed Distribution： 分布固定且正态

2.2 Techniques for Testing Assumptions

First Glance:

  • Run sequence plot ( y = Y, x = index)
  • Lag plot ( y = Yt, x = Yt-1) | Autocorrelation Plot (ACF PACF)
  • Histogram
  • Normal Probability Plot (not useful) | PPCC (Probability Plot Correlation Coefficient Plot)
